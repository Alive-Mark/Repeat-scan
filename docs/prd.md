# 需求文档

## 1. 应用概述

### 1.1 应用名称
重复文件扫描工具

### 1.2 应用描述
一款高效快速准确的文件扫描工具,用于检测文件夹中的重复文件,支持相同内容扫描、相同文件名扫描、相似文件名扫描三种模式,其中相似文件名扫描支持自定义相似度。支持文件删除、移动、预览等完整管理功能。同时支持扫描空文件夹并进行删除操作。

## 2. 页面结构与功能说明

### 2.1 页面结构
- 扫描配置页
- 扫描结果页
- 历史记录页

### 2.2 扫描配置页

#### 2.2.1 文件夹选择
- 提供文件夹选择功能
- 显示已选择的文件夹路径
- 支持多文件夹同时选择

#### 2.2.2 扫描模式选择
- 相同内容扫描:检测文件内容完全相同的文件
- 相同文件名扫描:检测文件名完全相同的文件
- 相似文件名扫描:检测文件名相似的文件
  - 提供相似度设置选项,支持用户自定义相似度百分比
- 空文件夹扫描:检测指定文件夹中的空文件夹
- 支持多种模式组合扫描

#### 2.2.3 扫描选项
- 提供文件类型过滤选项
- 提供文件大小范围筛选
- 提供忽略隐藏文件选项

#### 2.2.4 开始扫描
- 提供开始扫描按钮
- 点击后进入扫描结果页

#### 2.2.5 历史记录入口
- 提供查看历史记录按钮
- 点击后进入历史记录页

### 2.3 扫描结果页

#### 2.3.1 扫描进度显示
- 显示扫描进度百分比
- 显示当前扫描状态
- 显示已扫描文件数量
- 提供暂停/继续扫描功能
- 提供取消扫描功能

#### 2.3.2 结果展示
- 按重复组分组展示扫描结果
- 每组显示:
  - 重复文件数量
  - 文件列表,包含文件名、文件路径、文件大小、修改时间
  - 文件缩略图(支持图片、视频等)
- 空文件夹结果展示:
  - 单独区域显示空文件夹列表
  - 每个空文件夹显示完整路径
  - 显示空文件夹总数
- 显示扫描总结:总文件数、重复文件数、空文件夹数、可释放空间
- 支持按文件大小、修改时间、文件名排序
- 支持搜索过滤功能

#### 2.3.3 文件操作功能
- 单个文件操作:
  - 删除文件
  - 移动文件到指定位置
  - 打开文件所在位置
  - 预览文件内容
- 批量操作:
  - 勾选多个文件进行批量删除
  - 勾选多个文件进行批量移动
  - 智能保留:每组自动保留一个文件,删除其余重复项
  - 全选/反选功能

#### 2.3.4 空文件夹操作功能
- 单个空文件夹操作:
  - 删除空文件夹
  - 打开文件夹所在位置
- 批量操作:
  - 勾选多个空文件夹进行批量删除
  - 全选/反选功能

#### 2.3.5 结果导出
- 提供导出扫描结果功能
- 支持导出格式:CSV、TXT、Excel
- 导出内容包含文件路径、大小、修改时间等信息
- 空文件夹信息同步导出

#### 2.3.6 操作功能
- 提供返回按钮,返回扫描配置页进行新的扫描
- 提供保存本次扫描记录功能

### 2.4 历史记录页

#### 2.4.1 记录列表
- 显示历史扫描记录列表
- 每条记录显示:扫描时间、扫描路径、重复文件数、空文件夹数、可释放空间
- 支持按时间排序

#### 2.4.2 记录操作
- 点击记录查看详细扫描结果
- 删除历史记录
- 清空所有历史记录

#### 2.4.3 返回功能
- 提供返回按钮,返回扫描配置页

## 3. 业务规则与逻辑

### 3.1 扫描规则
- 相同内容扫描:通过文件内容比对判断是否重复
- 相同文件名扫描:通过文件名完全匹配判断是否重复
- 相似文件名扫描:通过文件名相似度算法判断是否重复,相似度达到用户设定值即判定为相似
- 空文件夹扫描:检测不包含任何文件和子文件夹的文件夹
- 组合扫描:同时应用多种扫描模式,结果分别展示

### 3.2 相似度计算
- 相似度范围:0%-100%
- 默认相似度:80%

### 3.3 扫描范围
- 扫描选定文件夹及其所有子文件夹中的文件
- 支持同时扫描多个文件夹
- 空文件夹扫描包含所有层级的子文件夹

### 3.4 文件操作规则
- 删除操作需二次确认
- 删除后文件进入系统回收站
- 移动操作需选择目标文件夹
- 批量操作前需确认操作数量

### 3.5 空文件夹操作规则
- 删除空文件夹需二次确认
- 删除后文件夹进入系统回收站
- 批量删除前需确认删除数量

### 3.6 智能保留规则
- 优先保留修改时间最新的文件
- 若修改时间相同,保留路径最短的文件
- 若路径长度相同,保留第一个扫描到的文件

### 3.7 历史记录规则
- 扫描记录自动保存
- 历史记录保存扫描配置和结果摘要
- 记录数据持久化存储

## 4. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 未选择文件夹 | 提示用户选择文件夹 |
| 未选择扫描模式 | 提示用户选择扫描模式 |
| 选择的文件夹为空 | 提示文件夹为空,无文件可扫描 |
| 选择的文件夹无访问权限 | 提示无法访问该文件夹,跳过该文件夹 |
| 相似度输入非数字 | 提示输入有效的数字 |
| 相似度超出范围 | 提示相似度需在0-100之间 |
| 扫描过程中无重复文件 | 显示未发现重复文件 |
| 扫描过程中无空文件夹 | 显示未发现空文件夹 |
| 删除文件失败 | 提示删除失败原因(如文件被占用) |
| 删除空文件夹失败 | 提示删除失败原因(如文件夹被占用) |
| 移动文件失败 | 提示移动失败原因 |
| 文件预览不支持的格式 | 提示该格式暂不支持预览 |
| 导出结果失败 | 提示导出失败,请重试 |
| 扫描过程中文件被删除或移动 | 跳过该文件,继续扫描 |
| 扫描过程中文件夹被删除 | 跳过该文件夹,继续扫描 |
| 磁盘空间不足 | 提示磁盘空间不足,无法完成操作 |

## 5. 验收标准

1. 能够成功选择单个或多个文件夹并显示路径
2. 四种扫描模式均可正常选择,支持组合扫描
3. 相似文件名扫描模式下可自定义相似度
4. 文件类型过滤和大小筛选功能正常
5. 点击开始扫描后能正确执行扫描
6. 扫描过程中显示进度,支持暂停、继续、取消
7. 扫描完成后按重复组展示结果
8. 结果中正确显示文件名、路径、大小、修改时间
9. 支持的文件类型能正确显示缩略图
10. 空文件夹扫描结果正确显示在单独区域
11. 显示扫描总结信息,包含空文件夹数量
12. 单个文件删除、移动、预览功能正常
13. 批量操作功能正常,包括批量删除、批量移动
14. 智能保留功能按规则正确保留文件
15. 单个空文件夹删除功能正常
16. 批量删除空文件夹功能正常
17. 扫描结果可正确导出为CSV、TXT、Excel格式,包含空文件夹信息
18. 历史记录正确保存和显示,包含空文件夹数量
19. 可从历史记录查看详细结果
20. 可返回配置页进行新的扫描
21. 异常情况下给出明确提示
22. 文件和文件夹操作有二次确认机制
23. 数据持久化存储正常
